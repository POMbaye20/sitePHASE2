Propriétés CSS utilisées dans cette page

Décrivez leur usage et leurs valeurs possibles

  1. background: couleur du fond de l'élément
    • Background applat de couleur ou linear gradient
    • On peut mettre des noms de couleurs ex:white, pink ou des valeurs en couleur web ex.#fff(blanc) ou en rgba
  2. color: couleur de la police
    • Cela sert à définir la couleur que l'on veut pour la police ex: color: black, olive etc...
  3. float: effet flottant
    • Voici quelques exemples de float :
    • float : right; : pour le faire flotter à droite
    • float : left; : pour le faire flotter à gauche

    • float : inherit; :

    • float : none; : il ne flotte pas
  4. font-family: police
    • La font-family sert à définir la police dans notre texte
    • Exemple : ci-dessus, la font family est : font-family:Impact, Haettenschweiler, 'Arial Narrow Bold', sans-serif; cela se rédige comme cela
      •
  5. font-size: taille de la police
    • Ici, on pourra définir la taille de la police. exemple: ici, font-size : 30px;
  6. height: hauteur de l'élément (ou de bloc) : permet de définir une hauteur à un élément exemple : height : 30px;
  7. left: position à partir de la gauche de la fenêtre
    • Left sera pour mettre un espace entre le bloc du coté gauche de l'écran ou de la fenêtre exemple: left : 50px; le décale vers la droite
  8. line-height: hauteur de l'interligne : définit un espace entre les lignes ou entre 2 paragraphes exemple : line-height : 300px;
    • Mais peut aussi se rédiger en % ou en em
      •
  9. margin: marges extérieures
    • Exemple : margin : 0 auto; placera l'élément au centre du conteneur
      •
  10. padding: marges intérieures
    • Exemple : padding : 10px; rajoute vraiment une marge à l'élément à l'intérieur d'un bloc. Peut également être en pourcentage
  11. position: position de l'élément dans la page (on n'a utilisé que la valeur (fixed) pour l'instant)
  12. text-align: alignement du texte : permet d'orienter le texte à un endroit précis
      13. Exemple :
    • text align : center;
    • text align : right;
    • text align : left;
  13. text-decoration: changement des valeurs héritées (ici le souligné qui est standard pour les liens sous tous les navigateurs est supprimé)
  14. top: position à partir du haut de la fenêtre
  15. transition: temps de transition lors d'une animation (ici sur les liens de la barre de menus)
  16. width: largeur de l'élément par rapport à la taille de la fenêtre si on indique une valeur relative (%, em, ...)
